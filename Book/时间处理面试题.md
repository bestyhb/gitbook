* 怎样使用Calendar直接设置时间，能够使用函数式直接设置吗？
```Java
// 正确
Calendar calendar=Calendar.getInstance();
calendar.set(2023,8,4);
// 错误
// 不能这么设置，因为set方法为void返回，所以不能写为等式。
Calendar calendar=Calendar.getInstance().set(2023,8,4);
```
