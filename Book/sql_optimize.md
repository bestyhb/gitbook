## 大数据量表更新提交
``` SQL
-- 2000万行数据更新（全部更新完再提交）-- 长期占用表资源，影响系统
UPDATE T_2000W SET SAL=SAL+100;
COMMIT;
-- 2000W行数据分批提交，每1W行提交一次。
-- 游标%ROWCOUNT记录游标遍历到第几行。
DECLARE
CURSOR C_2000W IS SELECT * FROM T_2000W;
BEGIN
FOR V IN C_2000W LOOP
UPDATE T_2000W SET SAL=SAL+100 WHERE ID=V.ID;
IF MOD(C_2000W%ROWCOUNT, 10000)=0 THEN COMMIT;
END IF;
END LOOP;
COMMIT;
END;
```
