在GitBook项目根目录下创建或编辑book.json。

1.2.1.1.1. Plugin (插件)

"plugins": [
    "hide-element",
    "include-codeblock",
    "copy-code-button",
    "anchor-navigation-ex",
    "advanced-emoji",
    "emphasize"
]

1.2.1.1.2. Configuration (插件配置)

  • SUMMARY添加分级编号 
    "pluginsConfig":{
  "theme-default":{
      "showLevel": true
  }
}
  • SUMMARY隐藏Gitbook LOGO
    "pluginsConfig":{
  "hide-element":{
      "elements":[".gitbook-link"]
  }
}
  • Article代码块复制
    "pluginsConfig":{
  "include-codeblock": {
      "template": "ace",
      "theme": "monokai"
  }
}
